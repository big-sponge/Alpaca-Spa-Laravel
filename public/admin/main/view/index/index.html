介绍 Alpaca-spa.js是一款轻量的前端JS框架，提供前端路由功能，前端视图渲染功能，前端套页面功能。目的是用来提高web项目的开发效率，前后端分离开发，同时使前端代码结构更加整洁。Alpaca-spa.js 区别于其他框架的主要特点是轻巧灵活，学习成本低。框架没有复杂的概念与特性，几乎都是最基本的JavaScript语法，也就是说读者只要有JavaScript语言基础，就能很快学会使用 Alpaca-spa.js框架，这是一个后台管理系统的DEMO，使用PHP-Laravel（服务后台） + LimitLess（前台样式）+Alpaca-Spa（前台逻辑）开发完成，主要功能模块有登录、定时任务管理、用户管理、权限管理、个人信息管理等。在实际开发中可以根据具体的需求添加新的功能。 当前登录账号是一个测试帐号，权限只有浏览功能，没有编辑等修改功能
